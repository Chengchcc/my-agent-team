# 01 — Glossary

Unified terminology for the entire project. Core principle: **one concept, one name** across all packages and milestones.

Covers 7 categories: Core concepts (Turn, Thread, Agent Loop, Checkpoint, Interrupt, Resume, Fork, Streaming), L1 Protocols (Message, ContentBlock, ChatModel, Tool), L2 Runtime (run, collectStream), Adapter layer (AnthropicChatModel), Tools (tools-common), L3 Framework (createAgent, Agent, Plugin, Checkpointer, ContextManager, Logger), L5 Backend (Agent Pool, SSE, AgentStore, Runner, AgentSpec), L4 Harness (two forms, bootstrap protocol), plus testing and project conventions.
