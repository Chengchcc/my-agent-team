# M18.3 Spec — Project 实体化：从裸字符串外键到受管实体

> **Status:** Spec → Ready for implementation.
> **Baseline commit:** `origin/master` @ `75e12459`（M18.1 + M18.2 已落、已 push）。

## 目标摘要

把 `projectId` 从 Issue 表上的裸字符串提升为受管实体：Project 表 + CRUD + git 归属元数据 + 前端管理页，并把 Issue 创建从「填字符串」改成「选 Project」。

## 7 个 Patch

| Patch | 内容 | 影响文件 |
|-------|------|---------|
| P1 | Project 表迁移 (id 5010) | `migrations.ts` |
| P2 | domain / port / adapter | `features/project/` (新) |
| P3 | service (CRUD + exists + 删除保护) | `features/project/service.ts` (新) |
| P4 | HTTP 路由 + main.ts 接线 + projectExists 注入 | `router.ts`, `main.ts`, `features/project/http.ts` (新) |
| P5 | 前端 api 客户端 + Project 管理页 | `api.ts`, `projects/page.tsx` (新) |
| P6 | Issue 创建改「选 Project」 | `issues/page.tsx` |
| P7 | 测试 | `project/*.test.ts`, `issue/service.test.ts` |
